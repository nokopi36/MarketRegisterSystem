# MarketRegisterSystem
端末1台で完結させる電子マネーレジシステム

## コミットのメッセージ
コミットメッセージには最初にprefixを書いています。以下はprefix一覧です。
* add: 新規ファイル追加
* remove：ファイル削除
* update：機能修正
* feat: 新しい機能
* fix: バグの修正
* docs: ドキュメントのみの変更
* style: 空白、フォーマット、セミコロン追加など
* refactor: 仕様に影響がないコード改善(リファクタ)
* perf: パフォーマンス向上関連
* test: テスト関連
* chore: ビルド、補助ツール、ライブラリ関連
